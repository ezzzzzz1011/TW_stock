> **「從行情追蹤到現金流規劃，打造存股族的終極儀表板。」**
> 
> TW.stock.app 是一個基於 Python Streamlit 開發的個人化投資系統。不僅提供即時的台股與 ETF 報價，更專注於**精準的配息預估**、**歷史回測**與**資產視覺化**。透過雲端資料庫同步，讓你的投資策略與資產配置隨時跟著你走。

🔗 **線上系統連結：** [👉 點擊此處立即體驗 TW.stock.app](https://ez-stock-fhedmdgxgniyeyhfxexy3m.streamlit.app/)

## ✨ 核心亮點功能 (Core Features)

### 📊 1. 智慧型 ETF & 個股深度解析
* **🤖 智慧配息頻率偵測**：系統自動分析歷史除息日，精準判斷標的為「月配、季配、半年配或年配」。
* **🛡️ 健壯的配息演算法**：針對配息停發或漏發的情況，系統會自動進行 0 填充校正，確保年化報酬率試算之精準度。
* **💰 個股合理價試算器**：輸入 EPS 與自訂本益比 (PE)，秒速換算「便宜價、合理價、昂貴價」位階，輔助進出場判斷。

### 💼 2. 投資組合與「月月配」現金流管理
* **☁️ 雲端無縫同步**：結合 Google Sheets 資料庫與個人帳號登入系統，個人投資組合與關注清單自動雲端存檔，資料永不遺失。
* **📅 自動化領息排程月曆**：系統依據台股慣例自動推算發放日，並精準扣除 **2.11% 二代健保補充保費**（單筆股利 > 2 萬時自動觸發），呈現最真實的實領金額。
* **📈 現金流視覺化柱狀圖**：將預估入帳的股息轉化為直覺的「Plotly 互動式長條圖」，讓您秒懂領息旺季與空窗期。
* **🥧 資產配置與報酬分析**：自動化圖表顯示各持股市值比重，並計算投資組合的「整體平均殖利率」與「總報酬率」。
* **📥 靈活資料匯入**：支援 CSV 檔案一鍵匯入投資清單，快速建立個人化數位資產帳本。

### ⚔️ 3. 實戰策略與規劃工具
* **🌐 全球市場戰情室**：即時監控 S&P 500、費半、美 10 年債殖利率及台指期全等關鍵指標，掌握全球市場趨勢。
* **🔮 存股未來財富複利試算**：設定初始資金、每月投入與目標殖利率，輕鬆試算多年後的總資產與「每月被動收入」。
* **🥊 ETF 終極 PK 擂台**：支援兩檔標的並排對比價格、漲幅、配息頻率與實質殖利率，輔助最佳標的篩選。
* **⭐ 即時行情關注清單**：打造專屬股票池，畫面自動定時刷新最新報價與漲跌幅。

## 🛠️ 系統技術架構 (Tech Stack)

| 模組分類 | 使用技術 / 服務 | 說明 |
| :--- | :--- | :--- |
| **前端與 UI** | `Streamlit` | 快速建構互動式 Web 應用程式介面。 |
| **資料庫** | `Google Sheets` | 透過 `gspread` 與 GCP Service Account 實現輕量級雲端讀寫。 |
| **即時報價引擎** | `Fugle (富果) API` | 提供極低延遲的台股盤中即時報價數據。 |
| **歷史配息引擎** | `FinMind`, `Yahoo`, `HiStock` | **三引擎聯集備援**，確保配息資料最齊全。 |
| **數據處理** | `Pandas`, `Plotly Express` | 負責矩陣運算與高質感的互動式圖表渲染。 |
